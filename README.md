# CCV-Pay - Payment Integration

## Installation

You can install the package via composer:

```bash
composer require quibax/ccv-pay
```

## Usage

``` php
$skeleton = new QuibaX\CCVPay();
echo $skeleton->echoPhrase('Hello!');
```

### Testing

``` bash
composer test
```
